# ToDo-list-advanced


Tecnologias:
React.js
JavaScript
Tailwind.css
GitHub Desktop


Instruções:
Antes utilize o comando 'cd (noma da pasta)' para adentrar no arquivo 
Após de o comando "npm run dev" para o arquivo abrir um local host 
Utilize Ctrl+click no link gerado 'http://localhost:PortaDoSeuPc/'


Observações:
Não esqueca de verificar se a pasta "node_modules" exite no arquivo, caso não utilize o comando "npm install"!
